== Description ==

I Blank This is an I Like This button, except you can replace the text with anything you want.  For example, instead of saying I like this, you could say, This is awesome!, I never knew that, or, Eww! This is gross!; the choice is up to you!

It comes with a widget to display the top voted posts.

== Installation ==

1. Upload the directory `/i-blank-this/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
